# Xiangqi Qt v1.0.0

Initial public release.

## Highlights

- Local PySide6 Xiangqi desktop game.
- Two-player mode and AI mode.
- Five progressive AI difficulty levels.
- Responsive AI calculation using background threads.
- Colored move history with Chinese move notation.
- Single-file Windows executable build.

## Windows EXE

The release executable is built with PyInstaller as `Xiangqi-Qt.exe`.
It bundles Python, PySide6, and application dependencies, so users do not need to install Python or PySide6 separately.
